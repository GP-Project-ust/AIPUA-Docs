## 📖 نبذة عن المشروع

مشروع تخرج لتطوير **نموذج أولي (Prototype)** لمساعد ذكي مؤسسي يعتمد على نماذج اللغة الكبيرة (LLMs) لتقديم خدمات استعلام ذكي وتحليل بيانات وتوليد تقارير مخصصة لأي مؤسسة تعليمية أو إدارية.

> 📌 **المرجع**: تطوير وتوسعة للمشروع رقم 27 من قائمة المشاريع المعلنة

---

## ✨ الميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🤖 **معمارية الوكلاء المتعددين** | Supervisor + SQL + RAG + Analyzer Agents عبر LangGraph |
| 🗄️ **طبقة المشاهد المبسطة** | Database Views Layer لرفع دقة SQL إلى 90%+ |
| ⚡ **مرحلة تكامل ذكية** | Smart Onboarding لتخصيص النظام في 8 ساعات |
| 🔒 **أمان متعدد الطبقات** | Blacklist / Restricted / Masked / Public |
| 🧠 **نموذج لغوي واحد قابل للتبديل** | محلي (Ollama) أو سحابي (API) — من الـ Dashboard |
| 🌐 **دعم كامل للغة العربية** | RTL + فصحى + عامية |

## 📚 الوثيقة الرئيسية

### 📄 [Project Vision Document (v3.0)](Project_Vision_Document.md)

وثيقة شاملة تصف رؤية المشروع ومراحله، تحتوي على 9 أقسام رئيسية:

1. **المقدمة** — الغرض، النطاق، التعريفات
2. **وضعية المشروع** — المشكلة، الرؤية، المهمة
3. **أصحاب المصلحة** — الفئات المستفيدة وأولوياتهم
4. **نظرة عامة على المنتج** — الوصف، الافتراضات، القيود
5. **الميزات الرئيسية** — 10 ميزات رئيسية
6. **المعمارية على مستوى الرؤية** — المخططات والتصميم
7. **القيود** — التقنية، الزمنية، الأكاديمية، التشغيلية
8. **مراحل النظام** — التكامل + التشغيل + التقنيات + حالات الاستخدام
9. **الخلاصة**

## 🛠️ التقنيات المستخدمة

| الطبقة | التقنية |
|--------|---------|
| Frontend | Next.js 16 + React + TypeScript + Tailwind CSS |
| Backend | FastAPI (Python) + SQLAlchemy 2.0 |
| AI Engine | LangChain + LangGraph (Multi-Agent) |
| LLM | Qwen 2.5 (محلي) أو OpenAI GPT-4o (سحابي) |
| Vector DB | Qdrant |
| Database | PostgreSQL 16 |
| Cache | Redis 7 |
| Containerization | Docker + Docker Compose |
| CI/CD | GitHub Actions |
| Monitoring | Prometheus + Grafana + LangSmith |
